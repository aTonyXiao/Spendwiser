BaseBackend

BaseBackend()

Base backend class for different backend types Database functions are designed around the Firestore Collection/Document style - Collections contain Documents - Documents contain data and sometimes Collections For more reference: https://firebase.google.com/docs/firestore/data-model

# dbGet(location, conditions, callback)

This function gets the data of a database 'document' in JSON or the all of the data of the 'document' data of a collection where the callback is called for each document in the collection
Parameters:
Name Type Description
location string Location in the database in the form: 'COLLECTION.DOCUMENT.COLLECTION...'
conditions Array Conditions for what documents to select from a collection, array in the form of: [FIELD, OPERATOR, COMPARISON] (e.g. ["name", "==", "Something"] **see https://firebase.google.com/docs/firestore/query-data/queries)
callback function Function that will be invoked to give the caller the data in JSON

View Source network/backends/basebackend.js, line 70

Examples
appBackend.dbGet("experimental.exp2", (data) => {
    console.log(data);
});
// conditions/queries can be stacked by adding more parameters after each other (before the callback)
appBackend.dbGet("experimental", ["hello", "==", "what2"], (data) => {
  console.log(data);
});
